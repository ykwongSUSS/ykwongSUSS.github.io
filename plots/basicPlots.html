0.1 Basic Plots

Julia’s default Plots.jl is powerful enough to cater to many different plots, and we shall use it as our goto Plot engine for the following basic cases:

  1. scatterplots
  2. scatterlines
  3. barplot
  4. histogram
  5. boxplot
  6. curves

1.6 Curves

In these plots, a range of x is used instead of dataframe values. The smoothness of the curve depends on the “length” of the range. Where the functions are direct polynomials, the range values can be substituted to return paired values(x,f(x)) for plotting on 2D.

using Plots
x = range(-2,2, length=100)
f(x) = x^2 -5*x +2
g(x) = x^4+3

plot(x,[f,g],size=(400,300), 
  label=[L"f=x^2-5x+2" L"g=x^4+3"], lw=3)

However where, the function(s) has inset functions (such as sin(x)) then a broadcast (dot function) is used for each inset function.

using Plots
using LaTeXStrings

x = range(-2,2, length=100)
f(x)= x^4 + 2
g(x)= 3*sin.(8x+8)+10
plot(x,[f,g], size=(400,300), leg=false,lw=3)
annotate!(0,14,L"g(x)=3sin.(8x+8)")
annotate!(-1,17,L"f(x)=x^4+3")
