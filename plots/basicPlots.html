0.1 Basic Plots

Julia’s default Plots.jl is powerful enough to cater to many different plots, and we shall use it as our goto Plot engine for the following basic cases:

  1. scatterplots
  2. scatterlines
  3. barplot
  4. histogram
  5. boxplot

1.1 Scatter

using Plots

trees |> Table
scatter(trees.girth, trees.height,
  title="1. Scatter Plot",
  subtitle="Date: 20 May 2025",
  xlabel = "Girth",
  ylabel = "Height",
  group = trees.name,
  size = (600,400)
  )

1.3 BarPlot

# using Statistics
# AvgHeight histogram of different trees

# split-apply-combine

treesGp = groupby(trees, :name) 
treesAvg = combine(treesGp, :height => mean)
treesAvg  |> Table
Table with 2 columns and 3 rows:
     name     height_mean
   ┌─────────────────────
 1 │ Elm      52.68
 2 │ Oak      53.05
 3 │ Sequoia  65.1333
# plot barPlot by trees.name

plot(bar(treesAvg.name, treesAvg.height_mean, 
         color=[:lightblue,:lightgreen,:orange], 
         legend=false, size=(400,300),
         title="Average height of trees"))
