Q markdown

Quarto is a file format that is supported by RStudio and Positron (based on VSCode). The file extension for a Quarto document is .qmd .

Quarto allows markdown to be written directly. Markdown includes text formatting, images and mathematical symbols. You can include TeX in markdown using embedded $ or $$ tags (inline or display modes).

With code chunks, Quarto further supports both computational codes (R, Python and Julia) and descriptive codes (LaTeX, Mermaid and GraphViz). Type backslash / in Quarto to see the selections.

Quarto extensions also allow for special support such as Tikz.

Quarto output is usually written to HTML or PDF formats as per YAML instructions.

0.1 Quarto check

To find out what version, packages or tools are installed in your Quarto environment, the following commands are available at the terminal prompt.

  1. quarto check
  2. quarto tools
