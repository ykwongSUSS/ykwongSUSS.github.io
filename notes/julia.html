julia

Julia code chunks may be used “directly” without additional statements in the yaml. Julia code chunk may be indicated from quarto’s Visual mode /julia command.

1 Add Julia Packages

import Pkg; 
Pkg.add("Plots"); 
Pkg.add("Pipe");
VERSION
v"1.11.6"

2 Using Julia package

using Plots
plot(sin, 
     x->sin(2x), 
     0, 2π, 
     leg=false, 
     fill=(0,:lavender))
plot!(size=(300,200))

3 Using Pipe

using Pipe
# allows for _underscore as placeholder for var
@pipe pi |> round(_, digits=4) |> println
3.1416
