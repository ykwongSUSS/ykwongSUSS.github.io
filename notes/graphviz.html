1 .dot Code Chunk

Example code chunk with GraphViz(dot)

{dot}
//| fig-width: 4
//| fig-height: 2
digraph {
  rankdir = LR;
  node [shape=circle, fixedsize=true, width=0.7, fontname=roboto];
  edge [color=brown,minlen=2, fontname=roboto]
    
    A [fillcolor=lightblue, style=filled, label="nodeA"]
    B [fillcolor=yellow, style=filled, label="B"]
    D [shape=rect, fillcolor=orange, style=filled]
    A -> C [label=13]
    B -> C [label=12]
    C -> D [label=11]    
}

*Note: cell options #| for {r} ; //| for {dot} ; %%/ for {mermaid}

2.1 Simple digraph with labels

{dot}
//| fig-width: 4
//| fig-height: 2
digraph{
  rankdir=TD;
  node[shape=circle, fixedsize=true, width=0.5, fontname=roboto];
  edge[minlen=2]

   {rank=same; A,B,D}  
    A -> B [label=1]
    B -> D [label=1]
    
    A -> C [label=2]
    C -> D [label=3]
  }
}
